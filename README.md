Project Overview
This project is an individual submission for the CST2550 Reset Coursework.
It implements a Local Supermarket Management System designed for small shops that currently use manual paper-based records.
The system provides digital support for:

Product management

Supplier records

Stock control

Sales processing

Searching

Low‑stock alerts

Report generation

The goal is to improve accuracy, reduce manual effort, and support daily supermarket operations.

System Features
Product Management
The system allows adding, updating, and removing products.
Each product includes details such as:

Product ID

Name

Category

Supplier

Barcode

Price

Quantity

Expiry or restock date

Supplier Management
Suppliers can be added and linked to products.

Stock Control
The system tracks stock levels, updates quantities, and identifies low‑stock items.

Sales Processing
Sales transactions can be recorded, and stock is updated automatically.

Searching
The system supports multiple search methods, including:

Search by product name

Search by barcode

Search by category or supplier

Reports
The system can generate:

Low‑stock reports

Products by category

Supplier stock lists

Sales summaries
